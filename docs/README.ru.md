# MacUpdate

**Обновление всех приложений на Mac — бесплатно и без регистрации**

MacUpdate сканирует установленные приложения и проверяет обновления через **Sparkle**, **Homebrew Casks** и **Mac App Store** — всё в одном терминале.

> **60 000+ приложений** в базе данных. Полностью open source. Без регистрации, без трекинга, без ограничений.

![MacUpdate TUI](docs/screenshot.png)

---

## Возможности

- **🔍 Умное сканирование** — `/Applications` и `~/Applications`, чтение `Info.plist`
- **📡 Проверка Sparkle** — автоматическое получение appcast XML фидов
- **🍺 Homebrew Casks** — проверка через `brew outdated --cask`
- **🏪 Mac App Store** — проверка через `mas outdated`
- **⚡ Параллельная проверка** — все фиды проверяются одновременно
- **🎯 ARM/Intel** — автоматический выбор правильного фида для вашей архитектуры
- **📦 Удалённая база данных** — всегда актуальная база из GitHub
- **💾 Кэширование** — локальный кэш 24ч, работает офлайн
- **🎨 Красивый TUI** — цветные статусы, прогресс-бар, лог

## База данных

**60 000+ приложений macOS** включая:

| Поле | Описание |
|------|----------|
| Sparkle feed URLs | Прямые ссылки на appcast XML |
| ARM/Intel фиды | Фиды для конкретной архитектуры |
| Homebrew cask названия | Для интеграции с `brew upgrade --cask` |
| Правила игнорирования | Системные приложения, установщики и т.д. |
| Цифровые подписи | Проверка подлинности разработчика |
| Ссылки на релизы | Прямые ссылки на журнал изменений |

База загружается из репозитория при каждом запуске и кэшируется локально на 24 часа. Отправляйте PR для добавления приложений.

## Установка

### Через Homebrew (рекомендуется)

```bash
brew tap yakushevhk/tap
brew install --cask macbee
```

### Из исходников

```bash
git clone https://github.com/yakushevhk/macupdate.git
cd macupdate
go build -o macbee .
./macbee
```

### Требования

- **macOS**
- **Homebrew** (опционально, для cask интеграции)
- **mas-cli** (опционально, для App Store интеграции)

```bash
brew install mas
```

### Обновление

```bash
brew upgrade --cask macbee
```

|---------|----------|

## Как это работает

```
┌─────────────────────────────────────────────────┐
│  1. Сканирование /Applications + ~/Applications│
│     └─ Чтение Info.plist → BundleID, версия    │
│                                                 │
│  2. Загрузка базы данных (GitHub → кэш → офлайн)│
│     └─ Сопоставление BundleID → Feed URL        │
│                                                 │
│  3. Проверка обновлений (параллельно)           │
│     ├─ Sparkle: получение appcast XML           │
│     ├─ Homebrew: brew outdated --cask           │
│     └─ App Store: mas outdated                  │
│                                                 │
│  4. Сравнение версий и отображение              │
│     └─ Семантическое сравнение версий           │
│                                                 │
│  5. Обновление (опционально)                    │
│     ├─ Sparkle: curl + open DMG/ZIP             │
│     ├─ Homebrew: brew upgrade --cask            │
│     └─ App Store: mas upgrade                   │
└─────────────────────────────────────────────────┘
```
